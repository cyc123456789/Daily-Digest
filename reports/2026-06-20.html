ZERO-DAY / KEV ACTIVELY EXPLOITED Cisco SD-WAN Manager 零時差漏洞遭在野利用，可繞過驗證取得 root 執行權
CVE-2026-20245 CRITICAL：Cisco Catalyst SD-WAN Manager 存在未驗證 API 端點，攻擊者可在不提供任何憑證的情況下發送特製請求，觸發本地提權漏洞並取得 root 執行權限（Privilege Escalation to Root）。攻擊者一旦 root 進入，可完全操控受影響設備的路由策略、流量過濾規則，乃至側移至後端企業網路。

CISA 已將此漏洞列入已知被利用漏洞目錄（KEV），並對所有聯邦民事行政機構（FCEB）發布緊急修補指令，要求 72 小時內完成修補。受影響版本：SD-WAN Manager 20.x 及以下；廠商已釋出 20.15.1 版本修補程式。

F5 針對 NGINX Open Source 兩枚重大漏洞釋出安全更新：其一為 HTTP/3 QUIC 模組 Use-After-Free 漏洞，可觸發遠端任意程式碼執行；其二為 HTTP/2 CONTINUATION 幀處理邏輯瑕疵，可被利用觸發記憶體耗盡（「HTTP/2 炸彈」變種），使高流量 Web 伺服器在數秒內停止回應。地下論壇已出現針對 QUIC 漏洞的概念驗證（PoC）程式碼，預計 72 小時內可能見到自動化掃描與利用嘗試。
